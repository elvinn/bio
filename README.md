# Bio 生成器

使用 AI 快速生成你的专属个人介绍。

## 技术栈

本项目采用 **TypeScript** 和 **Next.js** 开发。

## 在线演示

体验在线演示： https://bio.elvinn.wiki/

## 快速开始

1. 下载代码

```shell
git clone https://github.com/elvinn/bio
```

2. 安装依赖

```shell
cd bio
pnpm install
```

3. 设置环境变量，将 `.env-example` 重命名为 `.env`

```shell
mv .env-example.env .env
```

4. 本地开发

```shell
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000) 进行预览。
